## 项目概述

### 品牌信息

| 项目       | 内容          |
| -------- | ----------- |
| **品牌名称** | 汇帮忙         |
| **品牌标语** | 一个能帮上小忙的工具箱 |
| **参考**   | 腾讯帮小忙       |

### 设计目标

- 类似"腾讯帮小忙"的工具型网站
- 首页展示工具卡片
- 点击卡片进入工具详情页
- 支持后续整合新工具

***

## 设计风格

| 项目        | 选择              | 说明              |
| --------- | --------------- | --------------- |
| **布局风格**  | 侧边栏导航 + 右侧网格卡片  | 类似腾讯帮小忙         |
| **配色风格**  | 活泼多彩风格          | 每个工具不同颜色        |
| **响应式**   | 只做桌面端           | 公司内部工具，主要在电脑上使用 |
| **UI 框架** | Tailwind CSS    | 现代风格，灵活易定制      |
| **图标库**   | Iconify 免费彩色图标库 | 大量免费彩色图标        |

## 整体布局

### 页面结构

```
┌─────────────────────────────────────────────────────────────────┐
│  [Logo]  汇帮忙          🔍 搜索框        汇帮忙工具已累计帮助XXX人次，节省XXX分钟  │
├──────────────┬──────────────────────────────────────────────────┤
│  侧边栏        │              右侧内容区域                         │
│              │                                                   │
│  📦 全部      │              ┌─────────┐  ┌─────────┐         │
│  📄 公文工具   │              │ 卡片1    │  │ 卡片2    │         │
│  🎫 铭牌工具   │              │         │  │         │         │
│              │              └─────────┘  └─────────┘         │
│              │                                                   │
└──────────────┴──────────────────────────────────────────────────┘
```

### 顶部搜索栏

- **位置**：固定在页面顶部，首页和工具页位置不变
- **左侧**：Logo + "汇帮忙"品牌名
- **中间**：搜索框
  - 支持工具名称搜索
  - 无输入时显示统计信息
- **右侧**：统计信息
  - "汇帮忙工具已累计帮助XXX人次"
  - "节省XXX分钟"
  - 公文格式优化：每次节省22分钟
  - 会议名牌生成：后续补充
  - 座位名牌生成：后续补充

### 侧边栏

- **默认选中**：全部
- **分类跟随**：点击工具卡片后，侧边栏自动选中该工具所在分类
- **分类列表**：
  - 全部
  - 公文工具
  - 铭牌工具
- **后续可扩展更多分类**

### 右侧内容区域

- **网格布局**：卡片式展示
- **卡片信息**：
  - 图标（彩色）
  - 工具名称
  - 工具简介
  - 使用次数（动态显示）

### 公文格式优化页面

| 阶段       | 按钮文字     | 状态文字   | 说明         |
| -------- | -------- | ------ | ---------- |
| **初始状态** | 上传       | -      | 未上传文件时     |

| **转换中**  | 转换中，无法点击 | 正在转换格式 | 文件正在处理中    |
| **转换完成** | 点击下载     | 已完成请下载 | 文件处理完成，可下载 |

| 功能       | 设计               |
| -------- | ---------------- |
| **上传区域** | 文件上传按钮（无预览）      |
| **状态文字** | 正在转换格式 → 已完成请下载  |
| **按钮交互** | 上传 → 开始转换 → 点击下载 |

## 技术栈

### 后端

- **框架**：Flask
- **服务器**：Gunicorn + Nginx
- **部署**：Linux 服务器

### 前端

- **框架**：纯 HTML + CSS + JavaScript
- **CSS**：Tailwind CSS
- **图标**：Iconify
- **交互**：原生 JavaScript

### 数据存储

- **使用次数**：简单文件存储或 SQLite（后续可升级）
- **累计帮助人次**：简单文件存储
- **累计节省时间**：简单文件存储
